## Курсы валют ЦБ РФ

- **Источник**: `https://www.cbr-xml-daily.ru/daily_json.js`.
- **Сервис**: `app/Services/CurrencyRateService.php` получает и кеширует нужные курсы на 5 минут, повторно используя последнее успешное значение при недоступности API. При недоступности HTTPS выполняется повторная попытка без проверки сертификата и fallback через `file_get_contents`.
- **Вью-композер**: в `App\Providers\AppServiceProvider` для `partials.admin.navbar` передаётся массив `currencyTicker` с курсами USD, EUR и CNY и отметкой времени последнего обновления.
- **Отображение**: в `resources/views/partials/admin/navbar.blade.php` значения выводятся в правой части шапки вместе с подписью «Обновлено N мин назад».

Обновление данных выполняется каждые 5 минут; при ошибке запроса пользователю показываются сохранённые курсы.
